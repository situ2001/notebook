# 基础

记录一些基础。React是个JS框架，用于方便从小且独立的代码即components开始构建用户界面

# Component

component有一些property如props, state。

props是创建这个对象时候传进去的，state可以认为是对象私有的properties

比如

class Foo extends React.Component {
    render() {
        return (
            <div>
                {this.props.value}
            </div>
        );
    }
}

const element = <Foo value="114514" />;

传了个value进去，props对象就含有value

其实上面这个JSX语句<Foo value="114514" />等价于下面这个

React.createElement(Foo, {
  value: "114514"
});

component除了可以声明为一个继承自React.Component的类，还可以是一个Function

function Foo(props) {
    return (
        <div>
            {this.props.value}
        </div>
    );
}

为了使react能定位render的时候有哪些element变化了，需要一个key，这是一个特殊的预留属性。可以在创建对象的时候定义一个。
