# 命题与联结词

  • 命题

一个陈述句，可以确定真假(与之相反的就是谬误了)。

  • 联结词

与命题相结合，就是用来修改命题的内容的。比如

PPP: 上海是一个大城市

¬P\lnot P¬P: 上海并不是一个大城市

联结词 英文 符号
否定 Negation ¬\lnot¬
合取 Conjunction ∧\land∧
析取 Disjunction ∨\lor∨
条件 Condition →\rightarrow→
双条件 If and only if ⇆\leftrightarrows⇆

怎么记之间的逻辑关系呢？

程序语言...否定就是NOT，合取就是AND，析取就是OR，剩下两个就看真值表

条件是善意的前提，P不行，Q也可能行（想一下P⇒QP\Rightarrow QP⇒Q)，双条件就是要一起，才会真，不然就是假

P Q P→\rightarrow→Q P⇆\leftrightarrows⇆Q

T F F F
F T T F
F F T T

# 命题公式

像这样子的

  1. ¬(P∧Q)\lnot (P\land Q)¬(P∧Q)
  2. (((P→Q)∧(Q→R))⇆(S⇆T))(((P\to Q)\land (Q\to R))\leftrightarrows (S\leftrightarrows T))(((P→Q)∧(Q→R))⇆(S⇆T))

# 命题定律

这个表可谓是非常重要的了，比如其中的德摩根律

命题定律 表达式
对合律 ¬¬P⇔P\lnot \lnot P\Leftrightarrow P¬¬P⇔P
幂等律 P∨P⇔P,P∧P⇔PP\lor P\Leftrightarrow P, P\land P\Leftrightarrow PP∨P⇔P,P∧P⇔P
结合律 (P∨Q)∨R⇔P∨(Q∨R)(P\lor Q)\lor R\Leftrightarrow P\lor (Q\lor R)(P∨Q)∨R⇔P∨(Q∨R)
(P∧Q)∧R⇔P∧(Q∧R)(P\land Q)\land R\Leftrightarrow P\land (Q\land R)(P∧Q)∧R⇔P∧(Q∧R)
交换律 P∨Q⇔Q∨PP\lor Q\Leftrightarrow Q\lor\ PP∨Q⇔Q∨P
P∧Q⇔Q∧PP\land Q\Leftrightarrow Q\land\ PP∧Q⇔Q∧P
分配律 P∨(Q∧R)⇔(P∨Q)∧(P∨R)P\lor(Q\land R)\Leftrightarrow(P\lor Q)\land(P\lor R)P∨(Q∧R)⇔(P∨Q)∧(P∨R)
R∧(Q∨R)⇔(P∧Q)∨(P∧R)R\land(Q\lor R)\Leftrightarrow (P\land Q)\lor(P\land R)R∧(Q∨R)⇔(P∧Q)∨(P∧R)
吸收律 P∨(P∧Q)⇔PP\lor (P\land Q)\Leftrightarrow PP∨(P∧Q)⇔P
P∧(P∨Q)⇔PP\land (P\lor Q)\Leftrightarrow PP∧(P∨Q)⇔P
德摩根律 ¬(P∧Q)⇔¬P∨¬Q\lnot(P\land Q)\Leftrightarrow \lnot P \lor \lnot Q¬(P∧Q)⇔¬P∨¬Q
¬(P∨Q)⇔¬P∧¬Q\lnot(P\lor Q)\Leftrightarrow \lnot P \land \lnot Q¬(P∨Q)⇔¬P∧¬Q
同一律 P∨F⇔P,P∧T⇔PP\lor F\Leftrightarrow P, P\land T\Leftrightarrow PP∨F⇔P,P∧T⇔P
零律 P∨T⇔T,P∧F⇔FP\lor T\Leftrightarrow T, P\land F\Leftrightarrow FP∨T⇔T,P∧F⇔F
否定律 P∨¬Q⇔T,P∧¬Q⇔FP\lor \lnot Q\Leftrightarrow T, P\land \lnot Q\Leftrightarrow FP∨¬Q⇔T,P∧¬Q⇔F
